## 1. 产品概述

### 1.1 一句话描述
用户上传建筑 IFC 文件 → 浏览器内实时 3D 渲染 → 选择元素导出到布局编辑器 → 运行 DES 仿真和优化。

### 1.2 解决的问题
当前用户必须在布局编辑器中**手动逐一添加元素**来构建设施布局。对于已有建筑 CAD/BIM 图纸的客户（如 ICA），这是重复劳动且容易出错。

### 1.3 目标用户
- **CAD/设施工程师**：维护建筑 BIM 模型，需要将模型导入仿真平台
- **运营分析师**：需要基于真实建筑布局运行仿真，而非从头搭建
- **项目经理**：需要快速评估现有设施的优化潜力

### 1.4 成功指标

| 指标 | 目标 |
|------|------|
| IFC 加载成功率 | ≥ 95%（标准 IFC 2x3/4 文件） |
| 元素自动分类准确率 | ≥ 70%（允许手动修正） |
| IFC → 布局编辑器转换时间 | < 5 分钟（含手动调整） |
| 用户满意度 | 比手动建模节省 ≥ 80% 时间 |

## 2. 用户故事

### US-1: 上传 IFC 文件
> 作为 CAD 工程师，我想上传建筑的 IFC 文件，以便在平台中查看 3D 模型。

**验收标准：**
- 支持拖拽或点击选择 .ifc / .ifczip 文件
- 显示加载进度条
- 加载完成后在 3D 场景中渲染模型
- 显示模型基本信息（文件名、元素数量、楼层数）
- 不支持的格式显示明确提示

### US-2: 浏览 3D 模型
> 作为分析师，我想在浏览器中旋转、缩放、浏览 BIM 模型，查看建筑结构。

**验收标准：**
- 鼠标旋转/缩放/平移
- 图层开关（建筑结构/门窗/空间/设备/MEP管线）
- 楼层切片滑块（选择查看哪一层）
- 点击元素显示属性信息

### US-3: 识别和分类元素
> 作为分析师，我想让系统自动识别 IFC 模型中的设施元素（入口、检查站、缓冲区等），以便快速建立仿真模型。

**验收标准：**
- 系统自动将 IFC 元素分类为：入口/出口/检查站/缓冲区/操作台
- 左侧显示分类后的元素列表
- 用户可以修改任何元素的分类
- 用户可以设置每个元素的容量和服务时间
- 标记哪些元素可移动/可调容量（用于优化）

### US-4: 导出到布局编辑器
> 作为分析师，我想将识别后的元素导出到布局编辑器，以便运行 DES 仿真和布局优化。

**验收标准：**
- 选择要导出的元素和楼层
- 系统自动生成元素间的路由（基于空间邻近性）
- 导出后自动保存为新的布局，并跳转到布局编辑器
- 在布局编辑器中可以看到所有导出的元素，位置与 BIM 模型一致
- 可以立即运行 DES 评估

### US-5: 保存和加载 BIM 项目
> 作为工程师，我想保存 BIM 导入的配置，以便下次继续工作。

**验收标准：**
- 保存 BIM 项目（元素分类、参数配置）到数据库
- 下次打开时可以加载已保存的项目
- BIM 元数据（文件名、楼层等）保留

## 4. 技术方案摘要

### 架构决策
- **纯前端方案**：使用 `web-ifc` (WASM) 在浏览器中解析和渲染 IFC
- **不使用后端处理**：避免 IfcOpenShell 的 C++ 编译依赖和 Docker 镜像膨胀
- **复用现有基础设施**：Three.js (已有)、布局 DB (V62, 已有)、布局编辑器 (已有)
- **轻量级定位**：本模块是数据提取工具，不是 3D 渲染引擎。高级渲染未来交给 FactVerse 平台

### 新增依赖
- `web-ifc` ^0.0.57 — WASM IFC 解析器 (~5MB)

### 不需要
- ❌ 后端新 API（复用现有 `/api/v1/layouts`）
- ❌ 数据库新表（复用 `facility_layouts.layout_data` JSONB）
- ❌ Docker 镜像修改
- ❌ `ifcopenshell` (Python)
- ❌ `web-ifc-three`（已停维护，直接用 web-ifc 原生 API）

### 文件大小限制
- 标准：200MB
- 推荐：< 100MB（最佳用户体验）

### 5.3 关键交互细节

**加载进度：**
- 进度条显示 WASM 初始化 → 文件解析 → 几何生成 → 渲染完成
- 每个阶段有文字提示

**元素选中：**
- 鼠标悬停：半透明高亮
- 单击：实线高亮 + 右侧属性面板更新
- Shift+点击：多选

**图层开关：**
- 即时生效（toggle mesh.visible）
- 带颜色指示条

**导出对话框：**
- 显示将导出的元素数量
- 可选：自动生成路由
- 可选：选择目标楼层
- 确认后跳转到布局编辑器
