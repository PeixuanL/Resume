# 供热设备手册摘要

> 本文档汇总供热系统常见设备的选型原则、运行参数、维护要点和故障处理方法，供现场运维和 RAG 检索使用。

---

## 一、循环水泵

### 1.1 选型原则

循环水泵是供热系统的核心动力设备，选型应遵循以下原则：

**流量计算：**
- 设计流量 G（m³/h）= Q / (ρ × c × ΔT)
  - Q：系统热负荷（kW）
  - ρ：水密度（≈1000 kg/m³）
  - c：水比热（≈4.186 kJ/(kg·°C)）
  - ΔT：设计供回水温差（°C）
- 实际选泵流量 = 设计流量 × 1.05～1.10（安全系数）

**扬程计算：**
- 泵扬程 H（m）= 管网总阻力损失 / 9.81
- 管网总阻力 = Σ(沿程阻力 + 局部阻力)
- 实际选泵扬程 = 计算扬程 × 1.10～1.15（安全系数）

**选型要点：**
- 优先选用节能型（IE3 及以上效率等级）电机
- 工作点应落在水泵高效区（η ≥ 最高效率的 85%）
- 大型系统宜配置 2 台并联（1 用 1 备）或 3 台（2 用 1 备）
- 变频调速泵可在流量 30%～100% 范围内高效运行
- 泵体材质：叶轮铸铁或铸钢；轴密封优先选用机械密封

### 1.2 主要运行参数

| 参数 | 典型范围 | 报警阈值 |
|------|---------|---------|
| 流量 | 设计值 ±10% | 偏差 >15% 报警 |
| 扬程 | 设计值 ±8% | 偏差 >12% 报警 |
| 电机电流 | ≤ 额定电流 | > 额定电流 10% 报警 |
| 轴承温度 | ≤ 75°C | > 80°C 报警，>90°C 停机 |
| 振动烈度 | ≤ 2.8 mm/s（7.5 kW以下）；≤ 4.5 mm/s（>7.5 kW） | 超标报警 |
| 噪音 | ≤ 85 dB(A)（距设备 1 m） | 明显异常时检查 |
| 运行温度（介质） | ≤ 120°C（热水泵） | 超过设计温度报警 |

### 1.3 维护保养规程

**日常检查（每次巡检）：**
- 检查运行电流、压力、流量是否在正常范围
- 检查轴承温度（手触摸不烫手，约 40～60°C 正常）
- 听声音，无异常噪声（金属摩擦、撞击声为异常）
- 检查机械密封处泄漏（允许轻微渗漏，不超过 3 滴/min）

**定期保养：**
- 每月：检查联轴器对中，紧固地脚螺栓
- 每季度：对球轴承加注润滑脂（用量：2/3 轴承室容积）
- 每半年：检查叶轮磨损，必要时更换磨损环
- 每年：拆泵检查，更换机械密封、O 形圈

### 1.4 常见故障及处理

**故障 1：流量不足（比设计值低 >15%）**
- 原因：叶轮磨损、汽蚀、管网阻力增大、阀门未全开
- 处理：检查吸入侧压力（防汽蚀）；检查阀门开度；检查叶轮磨损

**故障 2：扬程不足**
- 原因：转速下降（变频故障）、叶轮磨损、系统漏气
- 处理：检查变频器输出频率；检查系统充水是否彻底排气

**故障 3：电机过热**
- 原因：电流过大、环境温度高、冷却风扇故障、绕组绝缘下降
- 处理：测量三相电流是否平衡；检查风扇运转；测量绕组绝缘电阻（≥ 0.5 MΩ）

**故障 4：轴承异常噪声**
- 原因：轴承缺油或损坏、转子不平衡、联轴器对中偏差
- 处理：补充润滑脂；振动分析；重新对中联轴器

**故障 5：机械密封大量泄漏**
- 原因：密封面磨损、弹簧失效、介质含固体颗粒
- 处理：更换机械密封组件；加装过滤器保护密封

**故障 6：水泵气蚀（汽蚀）**
- 现象：噪声增大、流量和扬程下降、叶轮出现点蚀坑
- 原因：吸入侧压力低于该温度饱和蒸汽压力
- 处理：提高吸入侧系统压力；降低介质温度；降低水泵安装高度

## 二、换热器

### 2.1 板式换热器

**结构特点：**
- 由多片波纹形换热板片叠合而成，板间形成交替流道
- 传热系数高（3000～7000 W/(m²·K)），远优于管壳式换热器
- 体积小、重量轻、拆洗方便
- 工作温度：≤ 150°C（普通橡胶垫）；≤ 180°C（耐高温垫）
- 工作压力：≤ 1.6 MPa（普通型）；≤ 2.5 MPa（全焊接型）

**选型参数：**
- 换热面积 F（m²）= Q / (K × ΔTm)
  - Q：热负荷（W）
  - K：总传热系数（W/(m²·K)）
  - ΔTm：对数平均温差（°C）
- 流速：板间流速 0.1～0.5 m/s（过高影响寿命，过低结垢严重）
- 压降：一次侧和二次侧压降宜≤ 50 kPa

**清洗周期：**
- 清水软化处理系统：每 2～3 年拆洗一次
- 未软化系统（硬水区）：每 1～2 年拆洗一次
- 结垢严重时（换热效率下降 >10%）：立即安排清洗
- 化学清洗：使用质量分数 5%～8% 的磷酸或柠檬酸溶液循环清洗，30～60 min，再用清水冲洗

**故障判断：**
- 换热效率下降（出口温度低于设计值 >3°C）：板片结垢，需清洗
- 一次侧或二次侧压降增大（超过正常值 1.5 倍）：流道堵塞，需清洗
- 一次侧与二次侧介质互窜：板片破裂或垫片泄漏，需更换板片或垫片
- 外部泄漏：垫片老化失效，需更换垫片并重新压紧

### 2.2 管壳式换热器

**结构特点：**
- 壳程（一次网热媒）+ 管程（二次网被加热介质）
- 传热系数较低（1000～3000 W/(m²·K)），但承压能力强
- 工作压力：≤ 2.5 MPa（一般型）；≤ 6.3 MPa（高压型）
- 工作温度：≤ 350°C（碳钢管束）
- 适用于介质腐蚀性强、压力高或含固体颗粒的场合

**维护要点：**
- 管束清洗：每 2～3 年拆管束人工清洗或高压水清洗
- 腐蚀检查：每次检修应进行超声波测厚，管壁减薄 >20% 时更换
- 管板检查：重点检查管板与管子的胀接或焊接处有无裂纹
- 壳程冲洗：停运期间应将壳程介质排净，防止积水腐蚀

**故障判断：**
- 换热效率下降：管束外侧积垢（石灰水垢、腐蚀产物）
- 管束泄漏（一二次侧互串）：管壁腐蚀穿孔或胀接松动，堵管处理
- 壳程压降增大：折流板损坏，局部短路

## 三、调节阀

### 3.1 电动调节阀

**工作原理：**
- 由执行器（电动机 + 减速齿轮）驱动阀芯，根据控制信号（4～20 mA 或 0～10 V）调节开度。

**主要技术参数：**
- 控制信号：4～20 mA DC（标准）；部分型号支持 RS485/Modbus 通信
- 反馈信号：4～20 mA（阀位反馈）
- 动作时间：DN50 以下通常 30～60 s 全行程；DN100 以上通常 60～120 s
- 工作压差：差压 ΔP ≤ 允许关断压差（选型时需校核）
- 泄漏等级：一般为 IV 级（≤ 0.01% 额定容量）；高要求时选 VI 级
- 防护等级：户外安装应选 IP67 以上

**开度-流量特性：**
- 线性特性：流量与开度成正比，适用于压降比（S值）较大（S > 0.5）的场合
- 等百分比特性（对数特性）：单位开度增量引起的流量变化率恒定，适用于供热调节（S值较小）
- 快开特性：一般用于开关控制，不适合调节

**S 值选取：**
- S = 阀全开压降 / 管路总压降
- S < 0.3：管路阻力大，阀权度小，调节效果差；应增大阀前差压或重新设计
- S = 0.3～0.6：一般调节场合；推荐等百分比特性阀
- S > 0.6：阀门压降占主导；可选线性特性阀

**常见故障：**
- 阀门不动作：检查控制信号、执行器电源（AC220V）、行程开关
- 阀门到位后继续动作（过行程）：检查限位开关整定
- 阀位反馈不准确：校准反馈电位器（零位和满量程）
- 阀体泄漏（外漏）：更换阀杆填料
- 流量调节线性差：阀门气蚀或污堵，清洗或更换阀芯

### 3.2 气动调节阀

**工作原理：**
- 由气动执行器（气缸或气囊）驱动阀芯，通过气源压力（0.02～0.1 MPa）和定位器控制开度。

**主要技术参数：**
- 气源压力：0.14～0.7 MPa（减压后工作压力 0.02～0.1 MPa）
- 控制信号：定位器接收 4～20 mA 信号，转换为气信号控制执行器
- 动作方式：气开型（fail-close，断气关阀）；气关型（fail-open，断气开阀）
  - 供热系统一次网调节阀：宜选气开型（断气安全关断）
  - 旁通泄压阀：宜选气关型（断气全开泄压）
- 动作时间：通常比电动阀快（10～30 s）

**定位器维护：**
- 定期清洁定位器气路过滤器（每季度）
- 检查气源压力是否稳定（波动 >±0.02 MPa 影响控制精度）
- 定期校准定位器（零点和量程）

**常见故障：**
- 阀门振荡（hunting）：定位器增益过高；气源压力波动；减小增益或稳定气源
- 阀门卡涩：阀杆积垢或填料过紧；清洗阀杆，调整填料压紧力
- 气源泄漏：检查气管接头、气缸密封圈

## 四、补水泵与定压系统

### 4.1 定压方式

**膨胀水箱定压（开式）：**
- 水箱安装在系统最高点，利用水位自然定压
- 定压值 = 水箱安装高度 × 9.81 / 1000（MPa）+ 静压
- 优点：简单可靠；缺点：开式系统易腐蚀，不适用于高温系统

**补水泵定压（闭式）：**
- 补水泵向系统补水，通过压力控制阀维持定压点压力恒定
- 适用于闭式热水系统（最常用方式）
- 定压点：通常选在循环泵吸入口，定压值使系统最低点表压 ≥ 0.05 MPa

**气压罐定压（氮气）：**
- 密闭气压罐内预充氮气，系统膨胀时压缩氮气，收缩时氮气复位
- 不需要补水泵频繁启停，系统压力稳定
- 气压罐容积 V = 3～5 倍系统膨胀水量

### 4.2 补水泵选型

**流量计算：**
- 正常补水量：系统总水量的 1%/h（年漏水量指标）
- 泄漏工况补水量：需满足一定压降速率下补水能力
- 选泵流量通常为系统循环流量的 1%～3%

**扬程计算：**
- 泵扬程 = 系统最高工作压力 + 0.05 MPa（富余量）

**变频控制：**
- 补水泵配置变频器，根据系统压力 PID 调节转速
- 设定值：定压点目标压力（如 0.35 MPa）
- 死区：±0.02 MPa（避免频繁启停）
- 高压保护：超过上限压力 0.05 MPa 停泵并报警

### 4.3 运行参数监控

| 参数 | 正常范围 | 报警条件 |
|------|---------|---------|
| 定压点压力 | 设定值 ±0.02 MPa | 偏差 >0.05 MPa 报警 |
| 补水量（日） | <系统水量 1% | >2% 报警（可能漏水） |
| 补水泵电流 | ≤ 额定值 | >额定值 10% 报警 |
| 水箱液位（开式） | 1/3～2/3 量程 | <1/4 或 >3/4 报警 |
| 软化水硬度 | ≤ 0.03 mmol/L | >0.1 mmol/L 更换树脂 |

## 五、传感器与仪表

### 5.1 温度传感器

**类型与选用：**
- 铂电阻（Pt100/Pt1000）：精度高（±0.1～0.5°C），适用于 -200～600°C，供热系统常用
- 热电偶（K/T型）：适用于高温（>500°C），供热系统较少用
- 热敏电阻（NTC/PTC）：体积小，响应快，成本低，用于房间温度测量

**安装要求：**
- 插入深度：管道直径 <80 mm 时，插入 1/3～1/2 管径；直径 >80 mm 时，插入管中心
- 安装位置：避开弯头、阀门后方（5D 以外），逆流方向安装
- 保护套管：高温高压介质应使用套管（损失响应速度）
- 接线：两线制（有误差）；推荐三线制或四线制（补偿导线电阻误差）

**校验周期：**
- 供热用温度传感器：每年校验一次（供暖季前）
- 计量用温度传感器（热量计配套）：依法定计量检定周期（通常 2 年）
- 校验标准：采用标准温度计（精度优于被校传感器 3 倍）对比校验
- 允许误差：过程控制用 ±1°C；计量用 ±0.5°C

**异常判断：**
- 读数突变（大于 20°C/min）：可能是传感器断路（Pt100 断路输出超量程值）
- 读数固定不变：可能是短路（Pt100 短路输出 0°C 左右）
- 读数偏差固定（与其他传感器对比）：可能是接线电阻未补偿或传感器漂移
- 读数噪声大：可能是电气干扰，检查接地和屏蔽层

### 5.2 压力传感器

**类型与选用：**
- 压阻式（硅压阻）：成本低，精度中等（±0.5%FS），供热常用
- 扩散硅压力变送器：精度高（±0.1%FS），长期稳定性好
- 量程选取：工作压力为量程的 2/3 左右（避免过载同时保证精度）
- 输出信号：4～20 mA（标准）；部分支持 Hart、Modbus 数字通信

**安装要求：**
- 引压管向上安装（液体介质），避免积气
- 引压管向下安装（气体介质），避免积液
- 热水管道引压需安装冷凝盘管（U形管）散热，保护变送器
- 安装截止阀和三通阀（便于维护和校验时隔离）

**校验周期：**
- 供热用压力传感器：每年校验一次
- 法定计量仪表：依检定规程（JJG 882）每 2 年检定一次
- 校验方法：标准压力表对比法或活塞压力计法
- 允许误差：过程控制用 ±0.5%FS；计量用 ±0.1%FS

**异常判断：**
- 读数为零（低于量程下限）：引压管堵塞或截止阀关闭
- 读数为满量程：引压管开裂或变送器超量程损坏
- 读数波动剧烈：系统压力确实波动（水锤等）或电气干扰
- 两台变送器读数差异 >5%：校验误差或安装位置不同（液柱高度差）

### 5.3 流量传感器

**类型与选用：**

*电磁流量计：*
- 适用于导电液体（供热用水）
- 精度：±0.5%（高精度型）～±1%（普通型）
- 无运动部件，维护量小
- 最小前直管段：5D；后直管段：2D
- 不适用于气体或非导电介质

*超声波流量计：*
- 外夹式：安装简便，不停水可安装，精度较低（±2%）
- 插入式：精度较高（±1%），需停水安装
- 适用于大口径管道（DN300 以上性价比高）

*涡街流量计：*
- 适用于液体、气体、蒸汽
- 精度：±1%
- 对振动敏感，安装时需远离振动源

**校验周期：**
- 过程控制用流量计：每 2 年校验一次
- 热量计配套流量计：每 2 年强制检定（法定计量器具）
- 校验方法：在线比对（与标准流量计对比）或离线标准装置校验

**异常判断：**
- 电磁流量计读数为零：可能是管道未满流（含气泡）或电极污染
- 流量计读数明显偏小：可能是管道结垢导致实际管径减小
- 流量计读数波动大：管道振动、气蚀、液体含气量高
- 流量与泵运行状态不匹配：流量计本身故障，进行比对校验

### 5.4 热量表（能量计）

**组成：**
- 流量传感器（电磁式或超声式）
- 温度传感器对（供水、回水 Pt100）
- 计算器（积分仪）

**计量原理：**
- 热量 Q = ∫ G × ρ × c × ΔT dt
- 其中 G：体积流量，ΔT = 供水温度 - 回水温度

**检定周期：**
- 热量表属于法定计量器具，强制检定
- 检定周期：6 年（首次检定合格后）

**常见误差来源：**
- 温度传感器不配对（两支温度计读数偏差）
- 流量传感器积垢（计量偏小）
- 流量传感器安装不规范（未满足直管段要求）
- 计算器时钟误差（积分时间不准确）
