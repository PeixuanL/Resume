# Data Integration 数据集成用户指南

> 面向数据工程师和系统管理员的数据集成模块使用手册。

---

## 1. Overview — 概述

Data Integration 是 FactVerse 平台的数据集成模块，为多源数据接入提供：

- **连接器管理** — 创建和管理各类数据源连接器
- **数据同步** — 定时或实时同步外部数据
- **数据质量** — 自动数据质量检查与报告
- **映射配置** — 源数据字段到平台字段的映射
- **同步历史** — 完整的同步记录和错误追踪

**导航路径：** `/data-integration`

---

## 2. Connector List — 连接器列表

**导航路径：** `/data-integration/connectors`

**API：** `GET /api/v1/dfslite/connectors`

**核心功能：**

- **连接器一览**：查看所有已配置的数据源连接器
- **状态监控**：实时显示连接器状态（在线/离线/错误）
- **快速操作**：启用/禁用/测试连接/手动同步

**支持的连接器类型：**

| 类型 | 协议/方式 | 适用场景 |
|------|----------|----------|
| REST API | HTTP/HTTPS | 第三方系统 API 对接 |
| MQTT | MQTT 3.1/5.0 | IoT 传感器实时数据 |
| OPC-UA | OPC Unified Architecture | 工业控制系统 |
| Modbus | Modbus TCP/RTU | PLC、仪表 |
| BACnet | BACnet IP/MSTP | 楼宇自控系统 |
| CSV/Excel | 文件上传 | 批量历史数据导入 |
| Database | JDBC | 外部数据库同步 |

## 3. Connector Wizard — 新建连接器向导

**导航路径：** `/data-integration/connectors/new`

**API：** `POST /api/v1/dfslite/connectors`

**向导步骤：**

1. **选择类型**：从支持的连接器类型中选择
2. **配置连接**：填写连接参数（地址、端口、认证信息）
3. **测试连接**：验证连接是否成功
4. **字段映射**：配置源数据字段到平台标准字段的映射关系
5. **同步策略**：设置同步频率（实时/定时）和数据保留策略
6. **确认创建**：预览配置并创建连接器

## 4. Connector Detail — 连接器详情

**导航路径：** `/data-integration/connectors/{id}`

**API：** `GET /api/v1/dfslite/connectors/{id}`

**详情内容：**

- **基本信息**：连接器名称、类型、创建时间、状态
- **连接配置**：地址、端口、认证方式（密码已脱敏）
- **字段映射表**：源字段 → 目标字段的完整映射
- **同步统计**：总同步次数、成功率、最近同步时间、数据量
- **操作按钮**：编辑 / 测试连接 / 手动同步 / 禁用 / 删除

## 5. Sync History — 同步历史

**导航路径：** `/data-integration/sync-history`

**API：** `GET /api/v1/dfslite/sync-history`

**功能说明：**

- **同步记录列表**：按时间倒序显示所有同步任务记录
- **状态筛选**：成功 / 失败 / 部分成功 / 进行中
- **错误详情**：失败记录展示错误消息、失败行数、重试次数
- **数据量统计**：每次同步的新增/更新/跳过记录数

## 6. Quality Dashboard — 数据质量仪表盘

**导航路径：** `/data-integration/quality`

**API：** `GET /api/v1/dfslite/quality`

**质量指标：**

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 完整性 | 必填字段的填充率 | > 99% |
| 准确性 | 数据值在合理范围内的比例 | > 98% |
| 及时性 | 数据在预期时间内到达的比例 | > 95% |
| 一致性 | 跨数据源的同一实体数据一致性 | > 95% |

**质量检查规则：**

- 空值检查：必填字段不允许为空
- 范围检查：数值型字段在合理范围内（如温度 -40~100°C）
- 格式检查：日期、枚举等字段格式正确
- 重复检查：检测并标记重复记录
- 时序检查：传感器数据时间戳连续性

**使用场景：** 数据工程师每日查看数据质量仪表盘，及时发现和修复数据问题，确保平台分析结果可靠。

## 7. Dataset Center — 数据集中心

**导航路径：** `/data-integration/datasets`

**API：** `GET /api/v1/dfslite/datasets`

**功能说明：**

- **数据集列表**：管理所有已接入的数据集，含名称、数据源类型、记录数、更新时间
- **数据集详情** (`/data-integration/datasets/:id`)：单个数据集的完整信息——字段 schema、预览数据、数据量趋势
- **版本管理**：每次数据同步生成新版本，支持回滚至历史版本

**使用场景：** 数据工程师管理已接入的各类数据集，确保数据可用性。

---

## 8. Method Library — 方法库

**导航路径：** `/data-integration/methods`

**API：** `GET /api/v1/dfslite/methods`

**功能说明：**

- **方法模板**：预置 15+ 种连接配置模板（Siemens/Mosquitto/Kepware/PI System 等）
- **版本管理**：方法库版本化，每次修改生成新版本，支持版本对比和回滚
- **可测试性**：每个方法支持创建测试任务，验证配置正确性后再部署到生产连接器

**使用场景：** 运维工程师复用已有方法模板，快速创建新连接器，减少重复配置。

---

## 9. Fusion Task — 融合任务

**导航路径：** `/data-integration/fusion`

**API：** `POST /api/v1/dfslite/fusion`

**功能说明：**

- **跨源融合**：将多个数据源的数据按实体 ID 关联融合（如将设备台账 + 实时传感器 + 历史工单融合）
- **融合任务管理**：创建、调度、监控融合任务
- **融合日志**：每次融合的输入输出记录，方便追溯

**使用场景：** 需要综合多个数据源做联合分析时使用，如将设备运行数据与维护记录关联分析。

---

## 10. Governance Canvas — 数据治理画布

**导航路径：** `/data-integration/governance`

**API：** `GET /api/v1/dfslite/governance`

**功能说明：**

- **治理规则配置**：配置数据质量规则、保留策略、访问控制
- **治理任务详情** (`/data-integration/governance/:id`)：单个治理任务的下钻详情
- **Pipeline 调试**：支持在线调试融合 pipeline，查看中间结果，快速定位问题
- **Audit Trail**：所有治理操作的完整操作日志（who/when/what）

**使用场景：** 数据治理团队配置数据质量规则和保留策略，监控数据治理执行状态。

---

## 11. Review Queue — 审核队列

**导航路径：** `/data-integration/reviews`

**API：** `GET /api/v1/dfslite/reviews`

**功能说明：**

- **待审核任务**：需要人工审核的数据操作（如敏感字段映射变更、新数据源接入申请）
- **审核操作**：通过 / 驳回，附审核意见
- **审核历史**：已完成审核的记录存档

**使用场景：** 当连接器配置变更需要人工审核时（如新增含敏感字段的数据源），管理员在审核队列中审批。

---

## 12. Audit Trail — 审计追踪

**导航路径：** `/data-integration/audit`

**API：** `GET /api/v1/dfslite/audit`

**功能说明：**

- **完整操作日志**：所有数据集成操作的审计记录（创建/修改/删除/同步/审核）
- **筛选与搜索**：按操作类型、时间范围、操作者、数据源筛选
- **合规报告**：导出符合 SOC2/ISO 27001 要求的审计报告

**使用场景：** 安全合规团队定期审计数据接入操作，确保无未授权访问。

---

## 13. Metrics Dashboard — 指标看板

**导航路径：** `/data-integration/metrics`

**API：** `GET /api/v1/dfslite/metrics`

**功能说明：**

- **接入规模**：总连接器数、活跃连接器数、数据总量
- **同步健康**：同步成功率、平均同步时长、失败趋势
- **数据质量趋势**：各质量指标随时间变化趋势
- **资源消耗**：API 调用次数、带宽消耗、数据存储量
