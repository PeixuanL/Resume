# 空间导入 — 用户指南

## 功能介绍

空间导入功能帮助您将 CAD 车间图纸快速转换为数字化布局，自动识别墙体、门、窗、围栏等空间结构，并支持物流仿真优化。

---

## 快速上手

### 第 1 步：准备图纸

1. 在您的 CAD 软件（AutoCAD / 中望 / 浩辰）中打开车间图纸
2. 文件 → 另存为 → 选择 **DXF** 格式
3. 文件大小建议 ≤ 200MB（超大文件系统会自动引导图层选择）

**支持的 DXF 版本**：R12 ~ R2018（所有主流版本）

### 第 2 步：上传图纸

1. 打开 FactVerse → 侧边栏 → **Simulation & Optimization** → **Spatial Import**
2. 将 DXF 文件拖拽到上传区域，或点击选择文件
3. 等待上传和自动识别（通常 5-30 秒）

### 第 3 步：查看识别结果

上传完成后，您会看到：

- **2D 布局图**：墙体（蓝色）、门（绿色）、窗（黄色）、围栏（灰色）
- **图层开关**：可以显示/隐藏各类型对象
- **识别统计**：总对象数、自动识别率

### 第 4 步：校正（如需要）

点击任意对象查看属性面板：

- **修改类型**：如果门被误识别为窗，可以直接切换
- **调整参数**：墙厚、门宽、围栏高度等
- **删除误识别**：删除不需要的对象
- **置信度标记**：红色 = 低置信度（建议人工确认），绿色 = 高置信度

### 第 5 步：导出

- **导出 JSON**：下载 FacilityLayout 数据，供其他系统使用
- **导出到 Designer**：一键生成 FactVerse Designer 3D 场景

## 物流仿真（进阶）

### 放置物流元素

在识别完空间结构后，使用物流编辑器放置：
- 🛤️ AGV 路径
- 📦 货架/存储区
- 🔋 充电站
- 📋 拣选站
- 🚛 装卸口

### 运行仿真

点击 **Run Simulation**，系统将：
1. 自动生成通道网络（A* 路径规划）
2. 模拟 AGV 调度和运输任务
3. 检测通道拥堵和瓶颈
4. 输出 KPI：吞吐量、平均配送时间、AGV 利用率

### 优化布局

点击 **Optimize**，系统使用 NSGA-II 多目标优化算法，自动寻找：
- 最短总路径方案
- 最少拥堵方案
- 综合推荐方案

## 大文件处理

| 文件大小 | 系统行为 |
|---------|---------|
| ≤ 200MB | 直接解析，无需额外操作 |
| 200MB - 1GB | 系统先扫描图层目录，您选择需要的图层后再导入 |
| > 1GB | 请在 CAD 软件中按楼层/区域分别导出为多个文件 |

---

## 图层命名建议

为获得最佳自动识别效果，建议按以下规范命名 CAD 图层：

| 类型 | 推荐图层名 | 示例 |
|------|-----------|------|
| 墙体 | `A-WALL` / `WALL` / `S-WALL` | A-WALL-OUTER, WALL-INT |
| 门 | `DOOR` | DOOR-SINGLE, DOOR-FIRE |
| 窗 | `WINDOW` / `WIN` | WINDOW-EXT |
| 围栏 | `FENCE` / `BARRIER` | FENCE-SAFETY |
| AGV 路径 | `AGV` / `AGV-PATH` | AGV-MAIN |
| 货架 | `SHELF` / `RACK` | SHELF-A |
| 充电站 | `CHARGE` | CHARGE-01 |
| 设备 | `E-EQUIP` / `EQUIP` | E-CNC, EQUIP-PUMP |

如果图层名不规范也不用担心——您可以保存自定义模板，下次同类图纸自动化率会更高。

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 上传后显示"无法解析" | 确认文件是 .dxf 格式（不是 .dwg） |
| 识别结果很少 | 检查图层命名是否匹配，或手动配置识别模板 |
| 墙体没有被识别 | 可能是单线墙，系统会以较低置信度识别，检查"低置信度"图层开关 |
| 大文件上传失败 | 超过 1GB 请在 CAD 中分文件导出 |
| 想恢复之前的布局 | 侧边栏 → Spatial Import，系统自动保存每次导入结果 |
