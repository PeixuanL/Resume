# 需求背景

# 技术优化
 1. 完成DFS核心能力的“Agent化”API改造，为全面拥抱AI打下100%的基础.最终场景化：通过AI实现“自然语言找表”和“AI自动解释表”。
 2. 分析agent做数据分析,分析并确认各类算法执行过程可见。算法运行过程中服务器的问题（排队）,以及发布后实现业务实时监控的数据探索任务监控运行过程中的服务器问题.
 3. designer(3D场景编辑器)模拟数据量级太大了,存在很多高频业务价值低的数据.方案:客户端采集降频，服务端降频，重采样
 4. node-red优化为数据接入工具包AI实现.
 
# 产品功能
1. 新增数据源管理（数据源接入管理，数据源列表管理）
2. 优化孪生体数据绑定的交互
3. 新增参数标签和数据集标签
4. 新增模型管理,管理算法包
5. 数据管理
	1. 新增数据分类管理
	2. 新增数据输出
	3. 新增数据分层管理（冷温热）
