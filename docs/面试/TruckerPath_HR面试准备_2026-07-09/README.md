我不是传统 C 端社区 PM，但我是一个能理解复杂真实场景、能把用户工作流拆成产品方案、能用英文跨团队沟通，并愿意补齐 UGC / 增长能力的产品经理。

## 面试前 10 分钟速看顺序

1. 先看 `04_HR面试准备.md` 的「面试主线」和「高频问题」。
2. 再看 `06_HR高频问题_薪酬通勤离职与简历追问.md`。
3. 如果担心行为面试题，看 `07_宝洁八大问与行为面试题库.md`。
4. 再看 `05_英文句库与常见单词.md` 的万能句。
5. 如果被问产品理解，看 `03_产品信息_TruckerPath.md` 的「主动输出版本」。
6. 如果被问为什么匹配，看 `01_岗位信息与匹配判断.md`。
7. 如果只剩 10 分钟开口练习，看 `08_继续准备_2026-07-10_英文HR速练.md`。
8. 如果进入产品 / 业务面，先看 `10_产品经理介绍与项目经历介绍_2026-07-11.md`。
