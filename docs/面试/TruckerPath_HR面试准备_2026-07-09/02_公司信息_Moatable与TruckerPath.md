# 公司信息：Moatable 与 Trucker Path

> 用途：外籍 HR 面试中回答 “Why our company?”、“What do you know about us?”
> 注意：以下为公开信息整理，面试时不要像背百科，取 2-3 点自然表达即可。

## Moatable 是什么

Moatable 官网将自己定位为构建和规模化 AI-native SaaS 产品的公司，强调用运营经验和资金资源支持 portfolio companies 成长，并希望帮助这些公司成为各自行业里的领先 SaaS 公司。

可面试表达：

```text
From my research, Moatable focuses on building and scaling AI-native SaaS businesses. I think this is interesting because it is not only about one app, but also about applying technology and operational expertise to specific industries.
```

## Moatable 与 Trucker Path 的关系

Moatable 官网 portfolio 中列出 Trucker Path，并将其描述为北美长途卡车司机、经纪人、车队使用的移动应用，帮助司机提高生产力、效率和路上沟通便利性，同时连接货运公司与卡车运力。

可面试表达：

```text
I noticed that Trucker Path is one of Moatable’s portfolio companies. It serves a very specific industry with real operational problems, which is a type of product I find very meaningful.
```

## Trucker Path 是什么

Trucker Path 官网定位为面向 truckers and fleets 的平台，强调卡车司机安全、效率、长途驾驶舒适和便利。官网显示其自 2013 年上线以来有大量下载、月活用户、路线里程和月度 check-ins。

可面试表达：

```text
My understanding is that Trucker Path is a leading platform for North American truck drivers and fleets. It combines truck-safe navigation, trip planning, parking information, fuel prices, weigh station status, load board, and driver community.
```

## 公司使命

Trucker Path About 页面写到，其使命是通过先进技术和司机社区，让 freight transportation 更 fast、reliable、efficient。

面试可转化为：

```text
I like that the mission is very practical: making freight transportation faster, more reliable, and more efficient. This means the product value can be directly connected to drivers’ daily work, time, cost, and safety.
```

## 你应该强调的公司匹配点

1. 这是一个垂直行业产品，不是泛娱乐 C 端产品。
2. 用户价值很具体：省时间、少绕路、找到停车、降低不确定性、提高安全。
3. 产品依赖实时信息、位置数据和社区贡献，这与你做过的复杂业务、数据闭环和用户流程拆解有连接。
4. 公司有全球 / 北美业务语境，外籍 HR 面可能会关注英文沟通、跨文化协作、海外用户理解。

## 不建议说的话

- 不要说“我想从 B 端转 C 端，所以这个是机会”。容易显得把岗位当跳板。
- 不要说“我很懂美国卡车行业”。你目前不该 overclaim。
- 不要只强调 AI。这个岗位核心仍然是 App / 社区 / 地图导航 / 货运场景。
