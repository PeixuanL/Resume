## 吸收到当前 Trucker Path 面试准备的原则

- 准备痕迹要说出来：不是只背自我介绍，而是说明自己准备了公司/产品、岗位匹配、缺口和英文表达。
- 回答要像外企沟通：先给结论，讲清楚边界，再给一个具体例子，不绕、不硬卖。
- 面试里的“缺口”不要藏：没有 trucking / navigation / 成熟 UGC，就说没有；接着说明如何学习和迁移。
- 文化匹配不要只谈福利：强调 clear goals、trust、direct communication、timely feedback、ownership、regular check-ins。
- 业务问题要比泛 HR 更具体：从 Trucker Path 的路线、停车、油价、称重站、货源、司机社区切入。

## 用在 Tell me about yourself 的方式

- 只吸收句型，不照搬夸张表达。
- 优先保留真诚边界：背景不是直接对口，但有可迁移的产品能力。
- 自我介绍不堆岗位词，围绕 3 个钩子展开：真实场景、信息判断、推进落地。
- 2026-07-11 更新：把开头改成更自然的外企面试口吻，先承认 not a perfect one-to-one match，再讲 transferable parts。
