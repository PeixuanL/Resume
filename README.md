# Resume

李沛宣 · B 端 AI / 产品经理简历与面试材料仓库。

## 目录结构

```
Resume/
├── 草稿/                 # 可编辑源稿（HTML / docx）
│   ├── 当前/             # 正在迭代的主稿与分支稿
│   ├── 历史/             # 旧版、两页版、早期命名稿
│   └── 素材/             # 头像等共用资源
├── 完成稿/               # 导出 PDF，用于投递
│   ├── 当前投递/         # 对外使用的最新版本
│   └── 归档/             # 历史 PDF，仅备查
├── docs/                 # 说明、面试、学习笔记
│   ├── 简历与反馈/       # 修订说明、调整结论、进度
│   ├── 面试/             # 问答沉淀、公司向准备
│   ├── 项目与学习/       # Inspector / Agent 等项目材料
│   └── 协作规范/         # AI 共建公约与审核提示词
├── 模板/                 # 简历 HTML 框架
├── 参考资料/             # 大体积 PDF（默认不纳入 Git）
└── .obsidian/            # Obsidian 工作区配置
```

## 命名约定

- **草稿 HTML**：`李沛宣_岗位_简历_单页_版本说明.html`
- **完成稿 PDF**：`李沛宣_岗位_简历_日期或版本.pdf`
- Word 另存网页时会生成 `*_files/` 目录，与对应 HTML 放在同一文件夹内，勿单独挪动。
